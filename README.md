# 🍋 Lemon Memo Automation

檸檬家事後台自動化工具，整合以下功能：

* 📝 Memo 自動回填
* 📅 排班勾選（匯入檔）
* 🍋 檸檬人空檔勾選
* 🧹 清空排班
* 💳 ATM 對帳
* 🔄 清潔異動

支援：

* LemonClean 後台自動登入
* Google Sheet 整合
* 排班自動化
* 訂單備註回填
* ATM 對帳與發票開立
* 清潔異動金額試算與回寫

---

# 🚀 安裝

```bash
pip install -r requirements.txt
```

# 📝 Memo 自動回填

## 功能目的

自動將歷史訂單客服備註回填至新訂單，避免客服重複查詢歷史紀錄。

適用於：

* 舊客回購
* 定期客戶
* 搬家後再次預約
* 長期合作客戶

---

## 查詢方式

### By Google Sheet

依指定列號執行。

支援格式：

```text
2
2,3,5
10-20
2,5-8,12
```

適合：

* Google Sheet 工作流程
* 指定訂單處理

---

### By 電話

輸入一支或多支電話：

```text
0912345678
0922333444
```

系統會：

1. 搜尋該客戶所有訂單
2. 找出未處理訂單
3. 比對最近可參照來源訂單
4. 產生回填預覽

適合：

* 客戶來電
* 單一客戶查詢

---

### By 搜尋條件

可依條件搜尋：

* 服務日期
* 購買日期
* 已付款
* 未付款

適合：

* 批次整理未處理訂單
* 每日客服作業

---

## 執行內容

系統自動：

1. 找出目標訂單
2. 比對同地址歷史訂單
3. 複製客服備註
4. 新成單帶入預設提醒文字
5. 更新服務狀態為已處理
6. 寫入執行紀錄

---

# 📅 排班勾選（匯入檔）

## 功能目的

透過 Excel 或 CSV 一次更新大量班表。

避免：

* 人工逐筆勾班
* 排班遺漏
* 重複操作

---

## 匯入格式

| 日期         | 類型 | 名稱  |
| ---------- | -- | --- |
| 2026-06-01 | 全8 | 王小明 |
| 2026-06-03 | 上4 | 陳小美 |
| 2026-06-05 | 清  | 王小明 |

---

## 支援班別

| 類型 | 說明     |
| -- | ------ |
| 全6 | 全天6小時  |
| 全8 | 全天8小時  |
| 上2 | 上午2小時  |
| 上3 | 上午3小時  |
| 上4 | 上午4小時  |
| 下2 | 下午2小時  |
| 下3 | 下午3小時  |
| 下4 | 下午4小時  |
| 晚2 | 晚上2小時  |
| 清  | 清除當天班表 |

---

## 執行內容

系統會：

* 搜尋專員 ID
* 讀取現有班表
* 合併新班表
* 自動處理衝突
* 一次送回後台

# 🍋 檸檬人空檔勾選

## 功能目的

快速找出可用檸檬人。

適合：

* 臨時派工
* 新成單安排
* 缺工補位

---

## 操作方式

輸入：

* 日期
* 班別

系統會自動檢查：

```text
檸檬人1
檸檬人2
檸檬人3
...
檸檬人10
```

---

## 執行內容

系統會：

1. 檢查班表
2. 檢查衝突時段
3. 找出第一位可用檸檬人
4. 可直接完成勾班

---

# 🧹 清空排班

## 功能目的

快速取消已安排班表。

適用情境：

* 客戶取消
* 大量改期
* 月排班重整

---

## 執行內容

指定日期後：

```text
全天
上午
下午
晚上
```

所有班別將被移除。

---

# 💳 ATM 對帳

## 功能目的

完成 ATM 收款後續作業。

適用：

* 每日財務對帳
* 補款確認
* 發票開立

---

## 自動執行流程

### Step 1

讀取 ATM Sheet

### Step 2

取得訂單編號

### Step 3

更新後台為已付款

### Step 4

開立電子發票

### Step 5

發送確認信

### Step 6

回寫 ATM Sheet

---

## 回寫欄位

| 欄位 | 說明     |
| -- | ------ |
| P  | 對帳完成時間 |
| Q  | 付款時間   |
| R  | 發票號碼   |
| S  | 確認信狀態  |
| T  | 系統更新狀態 |

---

## 適用人員

* 財務
* 區主管
* 營運管理

---

# 🔄 清潔異動

## 功能目的

協助計算與處理：

* 改期
* 取消
* 異動費
* 退款
* 車馬費
* 客訴退款

---

## 支援類型

### 車馬費發票

自動計算：

```text
專員數 × 100元
```

---

### 異動服務收款

依：

* 工作天數
* 客戶類型
* 工時
* 人數

自動試算異動費。

---

### 異動服務退款

自動計算：

```text
原訂單金額
- 異動費
= 應退款金額
```

---

### 客訴退款

協助建立退款資料。

---

### 物損退款

協助建立賠償退款資料。

---

## 階段 A：試算

系統自動：

* 查詢訂單
* 計算工時
* 計算異動費
* 計算退款金額
* 寫入清潔異動工作表

---

## 階段 B：同步

系統自動：

* 讀取待處理資料
* 更新後台資料
* 更新付款資訊
* 更新退款資訊
* 更新發票資訊
* 更新處理狀態
