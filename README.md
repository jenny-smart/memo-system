# 🍋 檸檬營運自動化工具

Lemon Operations Automation

整合客服、排班、財務與服務異動流程的營運工具。

---

# 功能總覽

## 📝 訂單客服備註

自動查詢歷史訂單並回填客服備註。

適用：

* 舊客回購
* 定期客戶
* 新成單整理

---

## 📅 排班功能

包含：

* 排班匯入
* 檸檬人空檔查詢
* 清空排班

適用：

* 月排班
* 臨時派工
* 補人力
* 排班重整

---

## 💳 ATM 對帳

包含：

* 待付款清單查詢
* 銀行明細配對
* 系統對帳更新

適用：

* 每日財務對帳
* ATM 收款確認

---

## 🔄 服務異動

包含：

* 異動費試算
* 車馬費處理
* 退款處理
* 後台同步

適用：

* 改期
* 取消
* 客訴退款
* 物損退款

# 📝 訂單客服備註

## 功能說明

自動將歷史客服備註回填至新訂單。

---

## 使用方式

### By Google Sheet

輸入列號：

```text
2
2,3,5
10-20
2,5-10
```

執行流程：

1. 輸入列號
2. 查詢訂單
3. 勾選要執行資料
4. 執行回填

---

### By 電話

輸入：

```text
0912345678
0922333444
```

執行流程：

1. 搜尋客戶訂單
2. 找出未處理訂單
3. 比對歷史訂單
4. 執行回填

---

### By 搜尋條件

條件：

* 服務日期
* 購買日期
* 已付款
* 未付款

執行流程：

1. 設定條件
2. 搜尋訂單
3. 勾選資料
4. 執行回填

---

# 📅 排班功能

## 1. 排班匯入

### 功能說明

批次更新專員班表。

---

### 檔案格式

| 日期         | 類型 | 名稱  |
| ---------- | -- | --- |
| 2026-06-01 | 全8 | 王小明 |
| 2026-06-02 | 上4 | 陳小美 |

---

### 班別代碼

| 代碼 | 說明    |
| -- | ----- |
| 全8 | 全天8小時 |
| 全6 | 全天6小時 |
| 上4 | 上午4小時 |
| 上3 | 上午3小時 |
| 上2 | 上午2小時 |
| 下4 | 下午4小時 |
| 下3 | 下午3小時 |
| 下2 | 下午2小時 |
| 晚2 | 晚上2小時 |
| 清  | 清除班表  |

---

### 執行步驟

1. 上傳 Excel / CSV
2. Dry Run 預覽
3. 確認結果
4. 正式儲存

---

## 2. 檸檬人空檔查詢

### 功能說明

快速找出可排班檸檬人。

---

### 執行步驟

1. 選擇日期
2. 選擇班別
3. 設定檸檬人數量
4. 點擊查詢

系統會：

* 搜尋檸檬人1~N
* 檢查班表衝突
* 顯示可用人員

---

## 3. 清空排班

### 功能說明

批次取消既有班表。

---

### 執行步驟

1. 輸入人員
2. 設定日期區間
3. 確認內容
4. 執行清空

注意：

⚠️ 執行後將直接更新後台

---

# 💳 ATM 對帳

建議依序執行。

---

## Step 1：待付款清單查詢

用途：

* 匯出 ATM 待付款訂單

執行步驟：

1. 選擇地區
2. 選擇日期
3. 執行查詢

結果：

* 寫入 ATM 工作表

---

## Step 2：配對銀行明細

用途：

* 配對匯款資料

執行步驟：

1. 貼上銀行明細
2. 輸入列號
3. 執行配對

結果：

* 自動配對訂單
* 更新 Sheet

---

## Step 3：更新系統對帳

用途：

* 完成付款流程

執行步驟：

1. 選擇地區
2. 輸入列號
3. 執行更新

系統將：

* 更新付款狀態
* 開立發票
* 發送確認信
* 回寫工作表

注意：

⚠️ 執行後無法復原

---

系統將：

* 更新收款資訊

* 更新發票資訊
* 更新處理狀態
